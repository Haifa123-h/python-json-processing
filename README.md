# Rick and Morty JSON Viewer

A Coursera project demonstrating how to store, retrieve, and process JSON data with Python.  
This app fetches character data from the [Rick and Morty API](https://rickandmortyapi.com/) and displays it in a Tkinter GUI.

---

## 🚀 Features
- Fetches character data (name, status, species, gender, origin, episodes).
- Displays character info in a Tkinter interface.
- Downloads and resizes character images using Pillow.
- Demonstrates JSON parsing and nested dictionary access in Python.

---

## 🛠️ Requirements
- Python 3.10+  
- Libraries:
  - `requests`
  - `pillow`
  - `tkinter` (comes with Python)
